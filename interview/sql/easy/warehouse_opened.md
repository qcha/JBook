# Работающие склады по городам

Задача с [Youtube](https://youtu.be/6WI_rsiD8Jw?t=2574) как задача с собеседования.

## Условие

Имеется таблица:

```sql
create table warehouse(
    warehouse_id  int primary key,
    name          text,
    city          text,
    date_open     date,
    date_close    date
);

insert into
    warehouse (warehouse_id, name, city, date_open, date_close)
values
    (1,  'Склад 1',  'Москва',            '2020-01-01', null),
    (2,  'Склад 2',  'Москва',            '2019-05-01', null),
    (3,  'Склад 3',  'Москва',            '2021-03-15', '2024-01-01'),
    (4,  'Склад 4',  'Санкт-Петербург',   '2018-07-01', null),
    (5,  'Склад 5',  'Санкт-Петербург',   '2022-11-01', '2025-01-01'),
    (6,  'Склад 6',  'Санкт-Петербург',   '2026-01-01', null),
    (7,  'Склад 7',  'Казань',            '2020-06-01', null),
    (8,  'Склад 8',  'Казань',            null,         null),
    (9,  'Склад 9',  'Новосибирск',       '2017-01-01', '2023-12-31'),
    (10, 'Склад 10', 'Новосибирск',       '2021-08-01', null);
```

Необходимо посчитать количество работающих складов на текущую дату по каждому городу, вывести только те города, количество складов в которых более 80.

Данные на выходе: город, количество работающих складов.

## Решение

Это стандартная задача на использование агрегатных функций, здесь самое главное - это не забыть, что date_close может быть проставлено как будущая дата закрытия (например, планируемая дата ликвидации склада).

```sql
select
    city,
    count(warehouse_id) as warehouse_count
from
    warehouse
where
    date_open <= current_date
and
    (date_close is null or date_close > current_date)
group by
    city
having
    count(warehouse_id) > 80;
```
